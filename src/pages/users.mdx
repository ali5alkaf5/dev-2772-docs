## The user model

The user model contains all the information about your users.

### Properties

<Properties>
  <Property name="_id" type="string">
    Unique identifier for the user.
  </Property>
  <Property name="name" type="string">
    The full name for the user.
  </Property>
  <Property name="email" type="string">
    The email for the user.
  </Property>
  <Property name="username" type="string">
    The username for the user.
  </Property>
  <Property name="password" type="string">
    The password for the user.
  </Property>
  <Property name="role_id" type="string">
    The role id for the user. 
  </Property>
  <Property name="invitedAt" type="datetime">
    Datetime of when the user was created.
  </Property>
  <Property name="invitedBy" type="string">

  </Property>
  <Property name="joinedAt" type="datetime">
    Datetime of when the user was joined.

  <Property name="updatedAt" type="datetime">
    Datetime of when the user was updated.
  </Property>
  <Property name="updatedBy" type="string">

## List all users {{ tag: 'GET', label: '/v1/users' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a paginated list of all your users. By default, a maximum of ten users are shown per page.

    # Request

    ### Optional attributes

    <Properties>
      <Property name="page" type="integer">
        The page number of users returned.
      </Property>
      <Property name="pageSize" type="integer">
        The page size number of users returned.
      </Property>
    </Properties>

    # Response 
    <Properties>
      <Property name="_id" type="string">

      <Property name="name" type="string">

        The email for the user.

        The username for the user.

        The password for the user.
      </Property>
      <Property name="role" type="object">
        The role for the user. 
      </Property>
      <Property name="role._id" type="string">
        Unique identifier for the role.

      <Property name="role.name" type="string">
        The name for the role.
